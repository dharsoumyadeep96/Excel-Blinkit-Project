This detailed  presentation highlights an end-to-end analytics project for the largest quick-commerce platform in India, Blinkit (formerly Grofers).  From $1.20 million in sales data representing 8,523 transactions across a 12-year period (2011-2022), the analysis provides strategic insights.

## Project Objective 

Conduct a thorough business intelligence analysis of Blinkit's inventory distribution, customer satisfaction, and sales performance.  The objective is to use Microsoft Excel's defined Key Performance Indicators (KPIs) and data visualizations to extract actionable insights and optimization opportunities.
